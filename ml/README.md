Step by Step Commands to Run the Flask Chatter Endpoint 

Check your Python Version
```
python3 --version
py --list
```

Create a virtual environment insice of ml/
```
py -3.11 -m venv .venv
.\.venv\Scripts\activate
```

Install the requirements.txt and try running main.py inside of .venv.
```
pip3 install -r requirements.txt
python3 main.py
```
